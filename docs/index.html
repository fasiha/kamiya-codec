This web app is based on the open-source Kamiya-Codec JavaScript/Node.js library which is in turn an implementation of the rules in two of Taeko Kamiya’s textbooks, The Handbook of Japanese Verbs (Kodansha) and The Handbook of Japanese Adjectives and Adverbs (Kodansha).

You may file an issue or get in touch.

Coders—if you pop open your browser’s JavaScript Dev Console, this library has been loaded as kamiya. Happy conjugating/deconjugating!
